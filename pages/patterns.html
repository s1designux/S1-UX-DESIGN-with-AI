S1 UX DESIGN GUIDE

Patterns

Core 컴포넌트(s1-btn · s1-table 등)를 조합한 반복 레이아웃 패턴. 신규 보여주기 컴포넌트를 만들지 않고 코어를 배치·조합한다.

v0.2

Modal

Pattern 2 sizes · Small / Medium 코어 재사용: Button · Table
ℹ️ Modal은 패턴이다. 닫기 아이콘(ic_닫기 원본)을 제외하고 내부 버튼·테이블은 모두 코어 컴포넌트(s1-btn · s1-table)를 그대로 배치한다. 코어 시각 스타일을 재정의(override)하지 않는다.

Small — 텍스트 확인 다이얼로그

짧은 안내·확인용 다이얼로그. width: 260px ~ 360px (콘텐츠 너비), 흰 배경 + 그림자(border 없음), radius 8px. 푸터는 코어 s1-btn-sm(xxsmall)을 우측 정렬로 배치한다. 

<!-- Small Modal — width 260~360px · border 없음 + 그림자 -->
<div class="s1-modal-overlay" hidden>
  <div class="s1-modal s1-modal--small" role="dialog" aria-modal="true">
    <div class="s1-modal-header">
      <span class="s1-modal-title">중복 안내</span>
      <button class="s1-modal-close" data-modal-close><!-- ic_닫기 --></button>
    </div>
    <div class="s1-modal-body">이미 등록된 항목입니다.</div>
    <div class="s1-modal-footer">
      <!-- 코어 s1-btn 재사용 (xxsmall=s1-btn-sm) -->
      <button class="s1-btn s1-btn-sm s1-btn-primary" data-modal-close>확인</button>
    </div>
  </div>
</div>
.s1-modal--small {
  min-width: 260px;       /* 최소 너비 */
  max-width: 360px;       /* 최대 너비 */
  width: max-content;       /* 260~360 사이 콘텐츠 너비 */
  gap: 32px;               /* section/lg */
  padding-top: 20px;       /* padding/block/md */
  padding-bottom: 20px;
  background: var(--modal-bg);            /* */
  border-radius: var(--modal-radius);    /* */
  box-shadow: 0 4px 16px var(--modal-overlay);  /* border 없음 — 그림자만 */
}
.s1-modal-overlay {
  position: fixed; inset: 0;
  background: var(--modal-overlay);       /* EX03 rgba 허용 */
  display: flex; align-items: center; justify-content: center;
}

본문 텍스트--modal-body-textvar(--color-text-secondary)

Medium — 콘텐츠 가변 (코어 Table 포함)

데이터 목록·폼 등 가변 콘텐츠용. width: 400px ~ 1200px, max-height: 70vh(본문 영역 스크롤), border 1px + radius 8px. 본문은 코어 s1-table을 실제 렌더하며 행 선택·버튼 클릭이 동작한다. 데이터 양에 따라 너비/높이가 확장된다. 

<!-- Medium Modal — width 400~1200px · max-height 70vh · border 1px -->
<div class="s1-modal-overlay" hidden>
  <div class="s1-modal s1-modal--medium" role="dialog" aria-modal="true">
    <div class="s1-modal-header">  <!-- List Header h52 -->
      <span class="s1-modal-title">항목 선택</span>
      <button class="s1-modal-close" data-modal-close><!-- ic_닫기 --></button>
    </div>
    <div class="s1-modal-body">
      <!-- 코어 s1-table 재사용 — 실제 렌더 · 행 선택 동작 -->
      <table class="s1-table s1-table--md"> ... </table>
    </div>
    <div class="s1-modal-footer">
      <button class="s1-btn s1-btn-secondary" data-modal-close>취소</button>
      <button class="s1-btn s1-btn-primary" data-modal-close>확인</button>
    </div>
  </div>
</div>
.s1-modal--medium {
  min-width: 400px;        /* 최소 너비 */
  max-width: 1200px;       /* 최대 너비 */
  width: fit-content;       /* 콘텐츠 양에 따라 확장 */
  max-height: 70vh;        /* 화면 70% 제한 */
  border: 1px solid var(--modal-border);  /* */
  overflow: hidden;          /* 본문 스크롤 radius 안쪽 격리 */
}
.s1-modal--medium .s1-modal-header {
  min-height: 52px;        /* List Header */
  padding: 16px 24px 8px;  /* pt16 / pb8 / px24 */
  border-bottom: 1px solid var(--modal-border);  /* */
}
.s1-modal--medium .s1-modal-body {
  padding: 16px 24px;
  overflow-y: auto;          /* 70vh 내부 스크롤 */
  flex: 1 1 auto;
}
