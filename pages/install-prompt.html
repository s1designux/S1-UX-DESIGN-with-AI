아래 CSS 토큰은 SW 디자인 시스템 V2.4의 공식 정의입니다.
이 토큰만 사용하여 UI를 구현해주세요.

---

### 핵심 규칙

1. **색상은 Semantic 토큰(--color-*) 사용**
   - Semantic → Foundation 구조. Semantic 레이어가 컴포넌트 단위까지 정의됨 (Variables 정본)
   - HEX 값 직접 사용 금지 (예외: rgba overlay)

2. **크기·간격은 Semantic 또는 Foundation 참조**
   - 여러 컴포넌트가 공유하는 값 → Semantic 경유
   - 컴포넌트 전용 값 → Foundation 직접 참조 가능

3. **Light/Dark 테마**
   - :root = Light (기본)
   - [data-theme="dark"] = Dark
   - 변수명은 동일, 값만 변경

4. **컴포넌트는 Semantic 토큰을 직접 사용**
   - 예: --color-button-bg-primary--default · --color-form-control-border-default · --color-table-cell-hover
   - 모든 Semantic 토큰은 Figma Variables 정본과 1:1 일치 (이름 같고 값 다르면 Variables가 정답)
