0개 항목 선택됨
기준 미등록 · UX 가이드 Figma 파일에서 컴포넌트 페이지/프레임 선택 후 1. 기준 등록 클릭
사용 방법
1단계. UX 가이드 Figma 파일을 열고 컴포넌트 정리 페이지/프레임 선택
→ 1. 기준 등록 클릭 (자동 저장, 한 번만 등록하면 됨)

2단계. 시안 파일로 전환 → 검수할 페이지/프레임 선택
→ 2. 교체 후보 스캔 클릭

3단계. 카드별 [교체] 또는 하단 [전체 교체] 클릭
→ 라이브러리에서 자동 import하여 swap 실행
※ 기준 컴포넌트가 라이브러리로 publish되어 있어야 다른 파일에서 교체 가능합니다.
