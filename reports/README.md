# Reports

이 폴더는 디자인 시스템 검수·리뷰 결과물을 보관한다.

| 파일 | 내용 | 상태 |
|------|------|------|
| `token-review.md`     | 토큰 구조 검증 결과 | 초안 |
| `component-review.md` | 컴포넌트 사양 검증 결과 | 초안 |
| `darkmode-review.md`  | 다크모드 candidate 항목 검토 | 초안 |
| `figma-map-review.md` | Figma 노드 매핑 검증 결과 | 초안 |
| `mvp0-setup-review.md`| MVP0 초기 구축 완료 리포트 | 완료 |

> 리뷰 결과는 `pages/md-review.html`의 리뷰 워크플로우와 연동된다.
> 확정된 항목은 registry JSON에 반영 후 status를 stable로 전환한다.
