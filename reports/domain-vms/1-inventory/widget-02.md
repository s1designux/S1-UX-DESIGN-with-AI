# 위젯 02. 일일 출입자 현황 — 재고조사

- 노드: 1965:14735 / 크기: 2902×1600
- 파일: 영상출입 통합관제 최종(KOR)_v0.0.6
- 스크린샷 관찰: 문그룹(출입문)별 일일 출입자 수를 임직원/내방객/협력사로 나눠 집계하는 관제 대시보드 위젯. 상단에 전체 합계 + 평균 대비 요약, 하단에 문그룹별 표(막대 포함)를 둔다. 같은 위젯의 **4가지 사이즈 변형(1×1 / 1×3 / 2×1 / 2×2)** 이 나란히 배치돼 있고, 별도로 오류 Alert(토스트) 1개가 캔버스 옆에 떨어져 있다. 숫자·막대 자리는 대부분 회색 placeholder 사각형(더미)이나, 일부 변형에 실제 예시 데이터 1행(C동 입구 300 등)이 들어 있다.

> **읽기 방법 표기:** 색은 get_variable_defs로 바인딩된 Variable 이름을 우선 기록(정본). Tailwind 코드의 `#RRGGBB`는 라이트-테마 fallback이라 실제 다크 렌더값과 다를 수 있어 참고로만 병기. 텍스트는 get_design_context 원문 verbatim.

## 반복/구조 관찰

1. **사이즈 변형 4종이 한 컴포넌트의 반응형 변형**(1×1 / 1×3 / 2×1 / 2×2). 위젯01(3종)보다 1종 많다(1×3 세로 긴 표 추가). 헤더는 동일 `Widget Header` 인스턴스를 4곳이 공유.
2. **반복 단위 = 표(Table)의 행(Row)**. 각 Row = [문그룹명 셀 + 전체(값+막대) 셀 + 임직원/내방객/협력사 셀]. 2×2는 여기에 평균/평균대비 2열이 붙음. 셀은 대부분 placeholder, 예시 1행만 실데이터.
3. **요약부 반복 = 방문 유형 3종(임직원·내방객·협력사)**. [라벨 + (숨김)아이콘 + count placeholder] 구조. 1×1만 색 점 범례(ellipse), 나머지는 아이콘(숨김) 방식.
4. **막대그래프(bar)**: 전체 셀 안의 가로 막대(`blue-dark/350`) + 평균 위치 dash 마커. 실데이터 행에서만 렌더됨(관찰).
5. **영역 구분**: ①Title container(위젯 제목 헤더, 최상단 2902폭 배너) ②각 위젯 본체 = 헤더 + 요약부 + 표 ③표 우측 scroll 바 ④Alert는 본체 밖 별도 오류 상태 요소.
6. **placeholder 일관**: 숫자·차트 자리는 `gray-dark/300`(#2e2f38) 회색 사각형(더미). 위젯01과 동일 색.
7. **아이콘 처리**: 요약부 유형 아이콘 3종(ic_계정/사용자/ID·ic_고객조회·ic_영업) 모두 hidden=true → 미표시. 표시 아이콘은 헤더 `ic_새로고침`, 1×1 Select Box caret `ic_화살표,더보기`, Alert `ic_주의(원형)`·`ic_닫기`.
8. **위젯01과 공유**: Widget Header(마스터 1024:33540)·Alert(구조/문구)·placeholder 색(gray-dark/300)·Title container 배너 패턴이 위젯01과 동일.

## 미확인 항목

- **차트/막대 placeholder의 최종 데이터**: 대부분 셀이 `gray-dark/300` 더미 → 실제 수치는 예시 1행(C동 입구 300/15/275/10) 외 미확인.
- **평균대비 지표의 색 규칙**: 2×2 변형 변수에 `red-dark/350`(#f06070)이 존재해 하락(음수) 강조용으로 추정되나, placeholder라 실제 적용/증감 방향 미확인.
- **1×3 표 헤더/셀 텍스트 색의 Variable**: 헤더 라벨(문그룹명 등) 개별 색은 design_context 미조회(2x1 실데이터 행에서 `body/secondary` 확인). 헤더 셀 색은 부분 미확인.
- **Alert wrapper(배경/보더) 색**: content(1965:14743)만 조회. 배경/보더/radius는 위젯01과 동일 추정이나 이 위젯 노드에서 직접 verbatim 미확인.
- **body/primary·secondary 색의 다크 실제값**: 코드 fallback(#202020/#353535)은 라이트 테마값. variable_defs는 `body/primary`=#ecedf0, `body/secondary`=#b8babf로 다크 해석 — 노드별 라이트/다크 최종 바인딩은 재확인 필요(Variable 이름은 확인됨).
- **Select Box(1×1)의 옵션/드롭다운 상태**: `전체 문그룹` 기본값만 확인, 열림/옵션 목록 상태는 시안에 없음 → 미확인.
