# 위젯05. 평균 출입자 수 현황 — 재고조사

- 파일: 영상출입 통합관제 최종(KOR)_v0.0.6
- 스크린샷 관찰: 출입문별 **평균 출입자 수(명/분)** 를 요약 지표 + 표 + 그래프(꺾은선/막대)로 보여주는 관제 대시보드 위젯. 위젯04보다 복잡해 **5개 변형**(1×1 / 1×2 / 2×1 A안 / 2×1 B안 / 2×2)이 배치돼 있고, 2×1은 **A안·B안 두 레이아웃 시안**이 나란히 있다. 큰 변형에는 **시간대별 평균 꺾은선 그래프**와 **출입문 선택 드롭다운(Select Box)**, 데이터 포인트 hover 툴팁이 포함된다. Alert(토스트)는 이 위젯엔 없다.

> **읽기 방법 표기:** 색은 get_variable_defs로 바인딩된 Variable 이름을 우선 기록(정본). Tailwind 코드의 `#RRGGBB`는 다크-테마 해석값 병기. 텍스트는 get_design_context 원문 verbatim. 그래프 선(Vector)은 SVG 스트로크라 Variable 미노출 → 색 미확인.

## 반복/구조 관찰

1. **5개 변형 = 3사이즈 + 2×1의 A/B 시안**. A안(테이블+꺾은선)과 B안(막대그래프+꺾은선)은 **같은 2×1 크기에 대한 두 레이아웃 제안** → Stage 2에서 택1/분리 판단 필요.
2. **핵심 반복 단위 3종**: ①데이터 행(출입문별, 4열: 출입문명/현재 평균/평균/평균 대비) ②꺾은선 그래프(시간대별 평균, 축+라인+툴팁) ③막대그래프(출입문별, B안 전용).
3. **테이블 골격은 위젯04와 동일**(Container gray-dark/50, Line divider 90°, placeholder gray-dark/300, 헤더 strong/데이터 default 라인). 컬럼만 다름(위젯04=장소명·혼잡도·대기자수·평균 ↔ 위젯05=출입문명·현재 평균·평균·평균 대비).
4. **혼잡도 표현 차이**: 위젯04는 3분할 막대 게이지, 위젯05는 요약블록의 `혼잡` 텍스트 badge + `평균 대비` 셀의 **추세 삼각형(Polygon)** 화살표. 위젯05 표엔 게이지 막대 없음(1×2 Row의 bar는 별개).
5. **Select Box(폼컨트롤) 재사용**: 꺾은선 그래프의 출입문 필터가 라이브러리 Select Box 인스턴스(110×28, form-control 토큰 바인딩) → core 폼컨트롤 재사용(관찰).
6. **위젯01과 공유**: Title container, Widget Header(마스터 1024:33540 동일 — 타임스탬프+새로고침 포함), 위젯 셸, placeholder 색. **위젯04와 공유**: 테이블 골격·라인 규칙·스크롤바·placeholder.
7. **hover 툴팁·Cursor**: 꺾은선 그래프에 데이터포인트 hover 툴팁(평균 11.0/14.0, 10시 00분)과 Cursor 목업 존재 — 위젯04의 hidden 시간대 툴팁과 유사한 인터랙션 목업(관찰).

## 미확인 항목

- **그래프 선·축·툴팁 색**: 꺾은선(Vector 25/26)·막대·축선(Vector 32)·추세 삼각형(Polygon)은 SVG 스트로크/도형이라 get_variable_defs에 색이 노출되지 않음 → **선/막대/포인트 색 전부 미확인**. (데이터 라인 강조색이 blue-dark 계열인지 별색인지 확인 불가.)
- **2×1 A안 vs B안 채택**: 같은 2×1 크기의 두 시안 중 무엇이 정본인지 시안상 미명시 → Stage 2 결정 필요(미확인).
- **1×2 테이블 Row의 컬럼-값 매핑**: Row에 막대(bar 60×10)+수치(300/275/10/15)가 섞여 있어 4개 헤더 컬럼과의 정확한 대응이 불명확 → 미확인.
- **혼잡 badge 텍스트/색**: 요약블록의 `혼잡` badge가 1×1 인스턴스에선 `(명/분)`로 표기되는 등 값이 변형별로 달라 실제 상태 라벨·색 매핑 미확인.
- **실제 데이터 값**: 대부분 셀이 placeholder(더미). 실제값은 샘플 행(C동 정문/0/0/-)과 그래프 목업 수치(3.1/1.6/11.0/14.0 등)만 존재 → 전체 데이터셋 미확인.
- **그래프 영역 컨테이너 배경/구조 상세**: 그래프 프레임의 개별 배경색·그리드라인 유무는 design_context 원문 미조회(축 라벨·라인 노드만 metadata로 확인) → 부분 미확인.
