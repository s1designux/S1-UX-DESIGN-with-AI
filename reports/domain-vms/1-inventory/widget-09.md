# 위젯 09. 실시간 영상 위젯 — 재고조사

- 노드: 1965:17109 / 크기: 2902×1600
- 파일: 영상출입 통합관제 최종(KOR)_v0.0.6
- 스크린샷 관찰: CCTV/실시간 영상 스트림을 **영상 타일(썸네일)** 로 배치하는 관제 대시보드 위젯. 같은 위젯의 **4가지 사이즈 변형(1×1 / 2×1 / 1×3 / 2×2)** 이 그리드 배치돼 있고, 각 변형은 영상 타일 개수·배치만 다르다(1개 / 좌우 2개 / 세로 3개 / 2×2 4개). 영상 영역은 모두 어두운 회색 placeholder(더미 썸네일)이며 각 타일 좌상단에 작은 회색 라벨 뱃지(카메라명 자리)가 있다. 상태 라벨은 대부분 `Default`, 2×2 변형만 `이상 발생`.

> **읽기 방법 표기:** 색은 get_variable_defs로 바인딩된 Variable 이름을 우선 기록(정본). 이 위젯은 다크 계열 토큰만 사용(base/black, gray-dark/*). 텍스트는 get_design_context 원문 verbatim. 헤더·아이콘·툴팁 등 hidden=true 요소도 verbatim 기록.

## "이상 발생" 상태 관찰 (2×2 변형)

- 2×2 변형만 상태 라벨이 `이상 발생`(1965:17216)이나, **get_variable_defs(1965:17181 전체) 결과에 red/status 계열 토큰이 전혀 없음** — 반환값은 `gray-dark/50`·`gray-dark/300`·`base/black`·`gray-dark/100`·`color/line/gray/subtle`(모두 회색/검정)뿐.
- 스크린샷상 2×2 그리드 바깥에 둥근 회색 테두리가 보이나(gray 계열), 타일 자체 색·라벨 뱃지 색은 Default 변형과 동일(`base/black` bg + `gray-dark/50` 썸네일 + `gray-dark/300` 뱃지).
- 즉 이 정적 시안에서 `이상 발생` 상태는 **색/토큰 차이로 표현돼 있지 않음**(라벨 텍스트로만 구분). 실제 이상 발생 시각 표현(예: 빨간 테두리·알림)은 이 시안에 없음 → 미확인.

## 반복/구조 관찰

1. **핵심 반복 단위 = 영상 타일(Container)**. 4개 사이즈 변형은 모두 동일한 타일을 개수·배치만 달리해 조합(1 / 2 가로 / 3 세로 / 4 그리드). 타일 = [base/black 컨테이너 + gray-dark/50 썸네일 영역 + 좌상단 gray-dark/300 라벨 뱃지].
2. **영상 그리드 레이아웃 규칙**: NxM 표기대로 열×행 그리드. 타일 간격 일정(~6px). 썸네일은 16:9 비율 유지(hidden 안내 텍스트로 명시).
3. **헤더·아이콘·툴팁이 전부 hidden**: 위젯 내부 헤더(실시간 영상/실시간 업데이트), 타일 오버레이 아이콘(확대·설정), 알고리즘 툴팁(출근 09:00~10:30) 모두 hidden=true → 실렌더에는 **영상 타일 그리드만** 보임. 이들은 인터랙션/호버 시 노출되는 요소로 추정(관찰).
4. **오버레이/툴팁은 1×1 위주**: ic_화면확대·ic_사용환경설정 오버레이는 1×1 타일에만, 알고리즘 툴팁은 1×1·2×1·1×3에만 존재(2×2 없음). 큰 그리드일수록 개별 타일 오버레이 생략(관찰).
5. **placeholder 일관**: 라벨 뱃지 `gray-dark/300`(#2e2f38), 썸네일 `gray-dark/50`(#131418). 위젯01·08과 동일 계열의 더미 placeholder 색.
6. **위젯08과 차이**: 위젯08은 리스트/필터 중심(탭·칩·검색·리스트행), 위젯09는 영상 타일 그리드 중심. 공통 헤더 처리 방식도 다름(08=Widget Header 인스턴스 표시, 09=Header area 프레임 hidden).

## 위젯01과 공유 구조 목록

- **Title container**: 2902폭 상단 배너 + 대제목 텍스트 — 위젯01과 동일 패턴.
- **placeholder 색 계열**: `gray-dark/300`(#2e2f38, 라벨 뱃지)·`gray-dark/50`(#131418, 썸네일) — 위젯01의 더미 placeholder 색 계열과 동일.
- **주석/상태 라벨 프레임**: `NxN` 크기 태그 + 상태 라벨(`Default`/`이상 발생`) 문서용 프레임 — 위젯01의 `[NxN size widget]` 관례와 동일 성격(단 위젯09는 상태 라벨을 별도 표기).

## 미확인 항목

- **`이상 발생` 상태의 실제 표현**: 2×2 변형이 `이상 발생` 라벨이나 red/status 토큰이 하나도 바인딩돼 있지 않음 → 이상 상태를 색/테두리로 어떻게 표현하는지 이 시안에 없음(라벨 텍스트로만 구분) → 미확인.
- **영상 썸네일 실제 콘텐츠**: 모든 썸네일이 `gray-dark/50` 더미 사각형(hidden 안내텍스트 "영상 썸네일 16:9 비율 유지"만) → 실제 영상/카메라 화면 없음 → 미확인.
- **라벨 뱃지(43×17) 내용**: 좌상단 회색 뱃지가 카메라명·시간·상태 중 무엇인지 텍스트 없음(placeholder) → 미확인.
- **헤더 텍스트 색 최종값**: Header area가 hidden이라 코드 fallback이 라이트값(#202020/#555)으로 나옴 — 실제 다크 렌더 시 바인딩 최종값 재확인 필요(단 hidden이라 렌더 안 됨).
- **오버레이 아이콘/툴팁 노출 조건**: ic_화면확대·설정·알고리즘 툴팁이 hover/hover-only인지, 특정 상태에서만 나타나는지 시안상 관계 미명시 → 미확인.
